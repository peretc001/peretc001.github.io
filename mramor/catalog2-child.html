В ассортименте большой выбор природного камня различный цветов, применяются различные виды обработки. У нас вы можете заказать изготовление по каталогу готовых работ. Также выполним работы по индивидуальным проектам

Для лестниц внутри помещения, где небольшая проходимость, в основном используется натуральный мрамор. В помещениях с большой проходимостью, применяется природный гранит.


Для наружных лестниц наиболее всего подходит камень со следующими видами обработки:


1. Полированный - создается гладкая поверхность с зеркальным блеском. Данный вид обработки отлично раскрывает рисунок камня. Дополнительно рекомендуется наносить специальный противоскользящий состав.


2. Шлифованный - создается равномерно шероховатая поверхность, неровность рельефа высотой до 0,5 мм. Чаще применяется для светлых пород камня, которые не имеют ярко выраженного рисунка.

3. Состаренный - камень обрабатывается специальными составами для достижения эффекта "антик", поверхность камня приобретает травленную структуру.


4. Лощеный - по результатам обработки поверхность получается гладкой, но без зеркального блеска. Хорошо открывает рисунок камня.
